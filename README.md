# AI Prompt Generator

A survey that generates a personalized system prompt for AI assistants (Claude, ChatGPT, Cursor, etc.)

## What it does

Asks ~20 questions about how you work and what you prefer, then spits out a system prompt you can copy-paste.

Topics covered:
- Profession and experience
- Thinking style (overthinker, perfectionist, etc.)
- Communication preferences (tone, verbosity, emojis)
- Decision making
- Output formatting
- Pet peeves

Questions adapt based on your profession (dev, lawyer, marketer, student, etc.)
